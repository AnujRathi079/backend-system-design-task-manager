# Backend System Design - Task Manager

This project demonstrates the design of a simple backend system for a Task Manager application.

The system includes:

- REST API Design
- GraphQL API Design
- gRPC Interface
- PostgreSQL Database Schema
- Authentication Strategy
- Docker PostgreSQL Setup

No application code is implemented. This project focuses on backend architecture and API design.

## Architecture

Client → API Layer (REST / GraphQL / gRPC) → PostgreSQL Database

## Technologies

- PostgreSQL
- Docker
- REST API
- GraphQL
- gRPC
- JWT Authentication
